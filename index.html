ZAGO Automation Project

ZAGO Automation Project

ZAGO Automation Project is a project that I develope at my free time during my work at Chain-reaction. The project is a web application that helps tester and manger to manage all the quality assurance process. the project is written in in diverse technologies such as: Node.js, React, flask, Express and more. The project is still in development and is not yet published. I hope this project will help testers and managers to work more efficiently and effectively. The project contain dashboard that shows the current status of the quality assurance process, the dashboard is updated in real time. In addition, the project contain a chats that help managers to have a better understanding of the current status of the quality assurance process. The project integrated with Jira and all the major testing frameworks such as: pytest, selenium, playwright and more.
